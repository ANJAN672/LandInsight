This project uses modern tools to ensure speed, security, and smart features:

- **Next.js**: The main framework used to build both the website and the server logic.
- **Node.js**: The runtime engine that powers the backend.
- **Gemini 2.5 AI**: The "brain" of the app. It analyzes land data and chats with you.
- **Leaflet & OpenStreetMap**: Used to show maps and let you draw land shapes.
- **Prisma**: A tool that helps the app talk to the database to save your land parcels.
- **PostgreSQL**: The database where all your saved land and chats are stored.
- **Tailwind CSS**: Used to make the app look premium and work on all screen sizes.
